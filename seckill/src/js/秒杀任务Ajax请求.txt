1、秒杀请求
http://m.made-in-china.com/comeIntoTheBowl   POST 请求  参数 activeKey(活动码) verCode(验证码) 
返回 字符串
10001  验证码错误
0: 非登录
1: 非高级供应商
2: 秒杀未开始
3: 开始
4: 等待
5: 结束
6: 失败
7: 网络异常
8: 成功
9: 不存在的秒杀任务



2、秒杀轮询状态请求 
http://m.made-in-china.com/comeIntoTheBowl/{activeKey} GET请求 
返回当前秒杀的当前状态，等待或成功或结束等  状态同上


2、服务器校时请求
http://m.made-in-china.com/validateTime/{activeKey}   GET 请求
返回 json
{
    time : 服务器时间字符串 (2015-11-11 11:11:11),
    remianTime : 字符串天数 或者 时分秒
    status : 任务状态(同秒杀请求)
}
